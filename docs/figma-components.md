## Typography Scale
- Heading 1: 36px / 2.25rem
- Heading 2: 30px / 1.875rem
- Heading 3: 24px / 1.5rem
- Body Large: 18px / 1.125rem
- Body: 16px / 1rem
- Body Small: 14px / 0.875rem
